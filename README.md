# FormForge

An Android library for building dynamic forms programmatically with ease.

## Features

- Declarative form building
- Built-in field validation
- Customizable field types
- Easy integration with existing Android projects

## Requirements

- Android SDK 21+
- Kotlin 1.8+

## Installation

Add the dependency to your `build.gradle`:

```groovy
dependencies {
    implementation 'com.wassimbeltaief:formforge:1.0.0'
}
```
